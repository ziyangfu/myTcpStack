# myTcpStack
一个采用C语言开发的用户空间TCP/IP协议栈，目的是学习如何写TCP/IP协议栈，以及加深对Linux TCP/IP协议栈的理解。

基本上是按照level-ip的教程和代码写的。
后续计划新开dev-C++分支，学习seastar中的DPDK + TCP/IP协议栈。
